## 🎯 立即开始

### 最简单的方式（推荐）

**Windows 用户：**
```bash
# 1. 进入项目目录
cd C:\Users\Administrator\.qclaw\workspace\dreamsync

# 2. 双击打开以下文件之一：
# - index.html (首页)
# - app.html (应用)

# 或者运行启动脚本：
start.bat
```

**Mac/Linux 用户：**
```bash
# 1. 进入项目目录
cd ~/.qclaw/workspace/dreamsync

# 2. 用浏览器打开：
open index.html
open app.html

# 或者运行启动脚本：
bash start.sh
```

## 📞 获取帮助

### 常见问题

**Q: 网页打不开？**
A: 确保使用现代浏览器（Chrome、Firefox、Safari、Edge）

**Q: 功能不工作？**
A: 刷新页面，检查浏览器控制台是否有错误

**Q: 如何启动完整服务器？**
A: 运行 `start.bat`（Windows）或 `start.sh`（Mac/Linux）

**Q: 数据会保存吗？**
A: 网页版使用 localStorage 本地保存，服务器版使用数据库
