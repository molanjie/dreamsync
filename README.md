# DreamSync - 梦境联网与共享平台

> "在梦境中相遇，在现实中相识"

DreamSync 是一款创新的梦境记录、分析和联网共享平台。用户可以记录自己的梦境，通过 AI 分析梦境内容，发现梦境模式，并与其他用户进行"梦境联机"——在相似梦境中相遇。

## ✨ 核心功能

### 🌙 梦境记录
- 语音/文字记录梦境
- 梦境标签分类（清醒梦、噩梦、预知梦等）
- 梦境清晰度评分
- 情绪分析
- 梦境人物/地点标记

### 🔮 AI 梦境分析
- 梦境主题识别
- 情绪倾向分析
- 梦境符号解读
- 重复梦境模式检测
- 睡眠质量关联分析

### 🌐 梦境联网
- 发现相似梦境
- 梦境匹配度算法
- 梦境共鸣（多人相似梦境）
- 梦境联机房间（实时共享梦境体验）
- 梦境社交图谱

### 👥 梦境社区
- 匿名分享梦境
- 梦境评论与讨论
- 梦境艺术家创作
- 梦境故事连载
- 梦境解梦互助

### 📊 个人梦境档案
- 梦境日历热力图
- 梦境统计图表
- 睡眠周期分析
- 梦境趋势预测
- 个人梦境词典

## 🛠️ 技术栈

### 后端
- Node.js + Express
- Socket.io (实时联机)
- SQLite3 + 向量数据库 (相似度搜索)
- Natural.js (NLP 处理)
- TensorFlow.js (梦境分析模型)

### 前端
- React 18 + Three.js (3D 梦境可视化)
- Tailwind CSS
- Web Speech API (语音输入)
- Canvas API (梦境绘画)
- WebRTC (梦境联机语音)

## 🚀 快速开始

```bash
# 克隆项目
git clone https://github.com/dreamsync/dreamsync.git
cd dreamsync

# 安装依赖
npm install
cd client && npm install && cd ..

# 初始化数据库
npm run init-db

# 启动开发服务器
npm run dev      # 后端 (端口 3001)
npm run client   # 前端 (端口 3000)
```

访问 http://localhost:3000

## 📝 使用指南

### 1. 记录梦境
- 醒来后立即打开 DreamSync
- 使用语音快速记录，或手写详细描述
- 标记梦境类型、清晰度、情绪
- 添加梦中出现的人物和地点

### 2. 查看分析
- AI 自动分析梦境主题和情绪
- 查看梦境符号的可能含义
- 发现重复出现的梦境模式

### 3. 发现共鸣
- 浏览社区分享的梦境
- 系统推荐相似梦境
- 与梦境相似的用户建立连接

### 4. 梦境联机
- 创建或加入梦境房间
- 设置共同的梦境主题或目标
- 使用引导音频同步入睡
- 醒来分享联机体验

### 5. 梦境创作
- 将梦境转化为视觉艺术
- 创作梦境故事
- 制作梦境音乐

## 🔒 隐私说明

- 所有梦境数据默认加密存储
- 分享梦境可选择匿名或实名
- 梦境联机采用端到端加密
- 用户完全控制自己的数据
