🍎 iPhone 安装

1

用 Safari 打开

请确保使用 Safari 浏览器打开此页面

2

点击分享按钮

点击 Safari 底部中间的「分享」图标 ⬆️

3

添加到主屏幕

在分享菜单中向下滑动，找到「添加至主屏幕」

4

点击「添加」

点击右上角「添加」，即可在主屏幕看到 DreamSync

🤖 Android 安装

1

用 Chrome 打开

使用 Chrome 或 Edge 浏览器打开

2

点击菜单

点击右上角三个点 ⋮ 选择「添加到主屏幕」

3

确认安装

点击「添加」，DreamSync 将出现在主屏幕
