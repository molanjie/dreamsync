# DreamSync - 梦境联网平台

> 在梦境中相遇，在现实中相识

## 🚀 快速开始

### 方式 1：直接打开网页（推荐）

```bash
# 进入项目目录
cd dreamsync

# 用浏览器打开以下文件：
# 1. 首页：index.html
# 2. 应用：app.html
```

**Windows 用户：**
```bash
# 直接双击打开文件，或者：
start index.html
start app.html
```

**Mac/Linux 用户：**
```bash
# 用默认浏览器打开
open index.html
open app.html
```

### 方式 2：启动完整服务器

```bash
# 安装依赖
npm install
cd client && npm install && cd ..

# 初始化数据库
npm run init-db

# 启动后端服务器（端口 3001）
npm run dev

# 新终端 - 启动前端（端口 3000）
npm run client
```

## 🎯 核心功能

### 1. 梦境记录
- 输入梦境标题和内容
- 选择梦境类型（普通梦、清醒梦、噩梦、重复梦）
- 评分清晰度（1-10）
- 一键保存

### 2. 梦境分析
- AI 自动识别梦境主题
- 情绪倾向分析
- 符号解读
- 相似梦境匹配

### 3. 社区互动
- 浏览他人分享的梦境
- 点赞和评论
- 发现梦境共鸣
- 匿名分享选项

### 4. 梦境联机
- 创建或加入梦境房间
- 实时聊天交流
- 多人同步入睡
- 分享梦境体验

### 5. 个人统计
- 梦境日历热力图
- 清晰度趋势
- 情绪分析
- 连续记录天数
